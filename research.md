# Tech Shop Fullstack Website — MVP Feature List

## Overview

A "Digital Showroom" for a refurbished laptop shop. The goal is to get the first 5 clients through the door fast — not to build a perfect marketplace. The website lists real units with real photos, battery health, and condition grades. Customers can browse, reserve, and walk in to test and buy. The seller never writes a single paragraph; structured data + auto-generated copy does the selling.

**MVP Goal:** Live in 1-2 weeks, 10-20 laptops listed, customer can find a laptop → see its real condition → contact/reserve in 2 clicks.

**Core Philosophy:** Sell laptops — doesn't matter if through the website directly or walk-ins generated by the site. The website is a spear, not a Swiss Army knife.

---

## PART 1: CPU & Specs Handling (Developer Architecture)

### Why Structure Matters

Do NOT give the seller a free-text "Processor" field. They will type "i7" and call it a day. Force structure so generation + full number are always captured.

### Admin Form Fields (CPU Specs)

| Field | Type | Example |
|---|---|---|
| `cpu_brand` | Dropdown | Intel / AMD |
| `cpu_model` | Dropdown (Dynamic) | Core i5, Core i7, Core i9, Ryzen 5, Ryzen 7 |
| `cpu_generation` | Dropdown | 8th Gen, 10th Gen, 11th Gen, 12th Gen |
| `cpu_full_number` | Text (Required) | i7-8565U |
| `cpu_cores` | Number | 4 |
| `cpu_base_clock` | Text (GHz) | 1.8 GHz |
| `cpu_boost_clock` | Text (GHz) | 4.6 GHz |

**Frontend Rule:** Always display `cpu_full_number` prominently (e.g., `Intel Core i7-8565U`) and always show the Generation badge next to it (e.g., `8th Gen`).

## PART 2: The "Little Speech" — Auto-Generated Copy

### The Developer's Solution

Do NOT give the seller a WYSIWYG "Description" box. They will leave it blank or write "Good laptop, works fine."

Instead, the seller fills structured fields → the website assembles the paragraph automatically using a template engine.

### Admin Form (What the Seller fills out)

| Field | Type | Seller enters |
|---|---|---|
| `condition_grade` | Dropdown | Excellent |
| `battery_health` | Number (0-100) | 87 |
| `battery_cycles` | Number | 220 |
| `physical_notes` | Small Textarea | "Minor shine on spacebar. Small scuff on right corner." |
| `refurb_actions` | Checkboxes (Multi-select) | ☑ New Thermal Paste, ☑ Fans Cleaned, ☑ Fresh Windows Install, ☑ Ports Tested |
| `included_items` | Checkboxes | ☑ Charger, ☑ Power Cable |

### Frontend Output (Auto-Generated)

> "This Excellent grade [Laptop Name] is powered by an Intel Core i7-8565U processor paired with 16GB of RAM. Our technicians have cleaned the fans and applied new thermal paste to ensure optimal cooling. The battery is currently holding 87% of its original capacity after 220 cycles — perfect for daily use. Minor cosmetic notes: Minor shine on spacebar. Comes with a genuine charger and a fresh Windows 11 installation. Walk in, test it, and drive it home today."

### Template Function (JavaScript Pseudocode)

```javascript
function generateSpeech(data) {
  let speech = `This ${data.grade} grade ${data.model} is powered by an Intel Core ${data.cpu_full} processor paired with ${data.ram}GB of RAM.`;

  if (data.refurb_actions.includes('thermal_paste')) {
    speech += ` We've applied new thermal paste and cleaned the fans.`;
  }

  speech += ` The battery holds ${data.battery_health}% capacity.`;

  if (data.physical_notes) {
    speech += ` Physical notes: ${data.physical_notes}`;
  }

  return speech;
}
```

### Trust Badges (Auto-Generated from Data)

| Input | Output |
|---|---|
| Battery % (e.g., 87) | 🟢 "87% — 220 Cycles" (color-coded circle gauge) |
| Grade (e.g., Excellent) | 🟡 "Excellent" badge |
| Refurb checkbox | ✅ "Thermal Paste Replaced" (green checkmark bullet) |
| Warranty | "6 Months Walk-in Warranty" (always visible) |

### Seller's Daily Workflow

1. Get a new laptop in the shop
2. Run Windows battery report → get battery % + cycle count
3. Take 5 photos with phone
4. Log into admin panel → fill structured form → upload photos → click "Publish"
5. Laptop appears on homepage + catalog within 1 minute
6. Seller never writes a single paragraph
